# Auditoria UI/UX/Design — Avant Charge Dashboard

## Problemas Identificados

### Global / Layout
1. Sidebar label "Navigation" genérico — deve ser "Zênite Tech" com logo
2. Sidebar muito larga por padrão (280px) — reduzir para 240px
3. Padding do conteúdo principal muito pequeno (p-4) — aumentar para p-6
4. Tipografia inconsistente entre páginas
5. Falta de separador visual entre seções da sidebar
6. Header do conteúdo sem borda inferior clara
7. Cards com fundo muito claro no modo claro — melhorar contraste
8. Botões de ação sem hierarquia visual clara

### Dashboard (Home)
1. Seção "Substituições de Termos" muito proeminente — deve ser colapsável
2. Dados de "Performance por Região Geográfica" são estáticos — marcar como tal
3. Cards de métricas do dia (Impressões, Cliques, etc.) com layout inconsistente
4. Seção "Performance de Sitelinks" com dados históricos — marcar como histórico
5. Comparação de Produtos com dados estáticos — marcar claramente
6. Ranking Top/Bottom com dados confusos (CTR muito baixo nos "top")
7. ROI/ROAS com valores negativos sem explicação clara
8. Insights da IA com texto "Recomendacao:" sem acento
